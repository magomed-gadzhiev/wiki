# 🚀 Серверы запущены!

## Статус

✅ **Backend (Django)** - запущен в фоновом режиме
- Адрес: http://localhost:8000
- API: http://localhost:8000/api/
- Admin: http://localhost:8000/admin

✅ **Frontend (Angular)** - запущен в фоновом режиме  
- Адрес: http://localhost:4200

## Что делать дальше?

1. **Откройте браузер** и перейдите на:
   ```
   http://localhost:4200
   ```

2. **Зарегистрируйтесь** или **войдите** в систему

3. **Создайте первую статью!**

## Остановка серверов

Для остановки серверов:
- Нажмите `Ctrl+C` в терминалах, где они запущены
- Или закройте терминалы

## Полезные ссылки

- Frontend: http://localhost:4200
- Backend API: http://localhost:8000/api/
- Django Admin: http://localhost:8000/admin

---

**Примечание:** Если серверы не запустились автоматически, запустите их вручную:

**Backend:**
```bash
cd backend
.\venv\Scripts\python.exe manage.py runserver
```

**Frontend:**
```bash
cd frontend
npm start
```
