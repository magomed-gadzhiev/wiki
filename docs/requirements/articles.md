# Требования к функционалу статей

## Основные возможности

### Создание и редактирование статей
- Создание новых статей с заголовком, кратким описанием и содержимым
- Редактирование существующих статей
- Поддержка визуального редактора GrapesJS с возможностью:
  - Визуальное редактирование с drag-and-drop интерфейсом
  - Форматирование текста (жирный, курсив, подчеркивание)
  - Заголовки разных уровней
  - Списки (нумерованные и маркированные)
  - Вставка изображений через Asset Manager
  - Вставка таблиц и layout блоков Bootstrap
  - Ссылки
  - Цвета текста и фона
  - Использование Bootstrap компонентов (контейнеры, строки, колонки)
  - Сохранение стилей и форматирования

### Импорт из Word
- Загрузка .doc и .docx файлов
- Автоматическая конвертация содержимого в HTML через Pandoc
- Fallback на mammoth если Pandoc недоступен
- Сохранение форматирования (параграфы, таблицы, стили, layout)
- Конвертация элементов в Bootstrap компоненты
- Сохранение нестандартных стилей через inline-стили
- Вставка конвертированного содержимого в редактор GrapesJS

### Изображения
- Загрузка изображений для статей
- Отображение изображений в галерее
- Альтернативный текст для изображений
- Автоматическая генерация URL для доступа к изображениям

### Публикация
- Статьи могут быть опубликованы или находиться в черновике
- Публикованные статьи видны всем авторизованным пользователям
- Черновики видны только автору и пользователям с правами просмотра
