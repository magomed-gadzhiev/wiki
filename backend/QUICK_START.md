## Следующие шаги:

### 1. Создайте суперпользователя (опционально, но рекомендуется)

```bash
cd backend
.\venv\Scripts\python.exe manage.py createsuperuser
```

Следуйте инструкциям для создания администратора.

### 2. Запустите Django сервер

```bash
cd backend
.\venv\Scripts\python.exe manage.py runserver
```

Backend будет доступен по адресу: http://localhost:8000

### 3. Установите Frontend

Откройте новый терминал:

```bash
cd frontend
npm install
npm start
```

Frontend будет доступен по адресу: http://localhost:4200

### 4. Начните использовать систему

1. Откройте http://localhost:4200
2. Зарегистрируйте нового пользователя или войдите как суперпользователь
3. Создайте первую статью!
