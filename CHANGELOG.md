# Changelog

## [1.0.0] - 2024

### Добавлено
- ✅ Полная система вики с редактированием статей
- ✅ Визуальный редактор GrapesJS с поддержкой Bootstrap компонентов
- ✅ Импорт содержимого из Word документов (.doc, .docx) через Pandoc
- ✅ Система прав доступа через группы пользователей
- ✅ Версионирование статей с возможностью восстановления
- ✅ Django Admin для управления контентом
- ✅ REST API с JWT аутентификацией
- ✅ Загрузка изображений для статей через Asset Manager
- ✅ Поиск и фильтрация статей по тегам и опциям
- ✅ История версий с описанием изменений
- ✅ Endpoint для получения списка пользователей
- ✅ Категории и разделы статей
- ✅ Теги и опции статей
- ✅ Автоматическая обработка 401 ошибок (разлогинивание и редирект на страницу авторизации)
- ✅ Обновление JWT токенов через refresh token

### Изменено
- ✅ Редактор статей: переход с Quill на GrapesJS для более гибкого визуального редактирования
- ✅ Импорт Word: использование Pandoc для более точной конвертации, fallback на mammoth
- ✅ Удалены все console.log/error/warn из production кода

### Технологии
- Backend: Django 5.0 + Django REST Framework
- Frontend: Angular v20, GrapesJS 0.22.13, Bootstrap 5.3.8
- База данных: SQLite
- Редактор: GrapesJS с плагинами (grapesjs-blocks-basic, grapesjs-table)
- Импорт Word: Pandoc (pypandoc), mammoth (fallback)
