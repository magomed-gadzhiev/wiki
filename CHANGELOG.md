# Changelog

## [1.0.0] - 2024

### Добавлено
- ✅ Полная система вики с редактированием статей
- ✅ Rich text редактор (Quill) с поддержкой изображений и таблиц
- ✅ Импорт содержимого из Word документов (.doc, .docx)
- ✅ Система прав доступа (просмотр, редактирование, удаление)
- ✅ Версионирование статей с возможностью восстановления
- ✅ Django Admin для управления контентом
- ✅ REST API с JWT аутентификацией
- ✅ Загрузка изображений для статей
- ✅ Поиск по статьям
- ✅ История версий с описанием изменений
- ✅ Endpoint для получения списка пользователей

### Технологии
- Backend: Django 5.0 + Django REST Framework
- Frontend: Angular v20
- База данных: SQLite
- Редактор: Quill
- Импорт Word: mammoth, python-docx
