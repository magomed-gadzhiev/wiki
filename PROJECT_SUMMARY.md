# Mikron Wiki - Итоговое описание проекта

## Реализованный функционал

### ✅ Backend (Django)
- **Модели данных:**
  - Article (статьи) с поддержкой прав доступа
  - ArticleVersion (версии статей) для версионирования
  - ArticleImage (изображения статей)
  
- **REST API:**
  - Аутентификация через JWT токены
  - CRUD операции для статей
  - Управление версиями статей
  - Загрузка изображений
  - Импорт из Word документов
  
- **Система прав доступа:**
  - Просмотр (can_view)
  - Редактирование (can_edit)
  - Удаление (can_delete)
  - Проверка прав на уровне API
  
- **Django Admin:**
  - Полное управление статьями, версиями и изображениями
  - Управление правами доступа

### ✅ Frontend (Angular v20)
- **Компоненты:**
  - Список статей с поиском и фильтрацией
  - Детальный просмотр статьи
  - Редактор статей с GrapesJS (визуальный редактор)
  - История версий с возможностью восстановления
  - Авторизация и регистрация
  - Категории и разделы статей
  - Теги и опции статей
  
- **Редактор статей (GrapesJS):**
  - Визуальное редактирование с drag-and-drop
  - Поддержка Bootstrap компонентов
  - Вставка изображений через Asset Manager
  - Вставка таблиц и layout блоков
  - Импорт из Word документов
  - Сохранение стилей и форматирования
  
- **Система прав:**
  - Отображение кнопок действий только для пользователей с правами
  - Проверка прав на фронтенде и бэкенде
  - Автоматическая обработка 401 ошибок (разлогинивание и редирект)

### ✅ Версионирование
- Автоматическое создание версий при изменении
- Просмотр истории версий
- Восстановление любой версии
- Описание изменений для каждой версии

### ✅ Импорт из Word
- Поддержка .doc и .docx файлов
- Конвертация в HTML с сохранением форматирования через Pandoc
- Обработка таблиц, параграфов, стилей и layout
- Fallback на mammoth если Pandoc недоступен
- Конвертация элементов в Bootstrap компоненты
- Сохранение нестандартных стилей через inline-стили

## Структура проекта

```
wiki/
├── backend/              # Django проект
│   ├── articles/         # Приложение статей
│   ├── users/            # Приложение пользователей
│   └── wiki_backend/     # Настройки проекта
├── frontend/             # Angular проект
│   └── src/
│       └── app/
│           ├── articles/ # Компоненты статей
│           ├── auth/    # Компоненты авторизации
│           └── core/     # Сервисы, модели, guards
└── docs/                 # Документация
```

## Технологии

- **Backend:** Django 5.0, Django REST Framework, SQLite
- **Frontend:** Angular v20, GrapesJS Editor, Bootstrap 5.3
- **Аутентификация:** JWT (Simple JWT) с автоматическим refresh
- **Импорт Word:** Pandoc (pypandoc), mammoth (fallback)
